# ONGC HR Applicant Tracking System (ATS)

An **HR-based Applicant Tracking System** developed exclusively for **ONGC** to streamline recruitment and applicant management.  
The system shifts traditional **Excel-based database processing** into a **structured, digital HR platform** with improved efficiency, scalability, and data consistency.
